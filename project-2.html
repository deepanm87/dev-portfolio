Project 2

A customizable e-commerce app with a Sanity CMS back-end that allows users to create and display products with their name, description and an image and to display if in stock or not.

Live Link

Project Overview

Users can create products to be displayed in store and we can add to basket, remove from basket, checkout via stripe payments and view order details of purchased orders.

Authentication via Clerk API using Google and email/password login. It also provides a custom backend using Sanity CMS to be able to add, edit and delete product data and view orders and categories of items. If an item's stock is updated to 0, it will not be available for purchase and is marked as out of stock. It also allows to search for products via keyword and allows to view items by category. It also provides a visual live editor in backend to make changes.
