Project 2

A recipes app that allows user to view and create recipes and a pantry page to store ingredients and allows passwordless login via magic link sent to email.

Live Link

Project Overview

Users can create recipes with the ingredient name and quantity, instructions, duration and attach a file with the image of the recipe and allows to edit, delete and the newly added recipes will be showcased on the home page.

Login is done using mailgun API service to allow passwordless login via email and also created a pantry page to show list of all ingredients with the ability to add, delete and edit them in shelves. A theme switcher to change the color theme is also provided and with the ability to logout.
